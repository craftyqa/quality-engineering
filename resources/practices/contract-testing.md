# Contract testing (practice)

**Contract testing** checks that services (or consumers and providers) agree on request/response shapes and behaviors—often **without** spinning up the full system. Use this page for **your** conventions, tooling choices, and links; align with [`integration-testing.md`](integration-testing.md) for where integration tests end and contracts begin.
